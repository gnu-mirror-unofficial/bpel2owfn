Reporting Bugs

If you find a bug in GNU BPEL2oWFN, please send us an electronic mail to bpel2owfn-bug@gnu.org. Include the version number which you can find by running bpel2owfn --version. Also include in your message the input BPEL process and the output that the program produced. We will try to answer your mail within a week.

You can also use this e-mail address to send us any questions, comments or suggestions about GNU BPEL2oWFN.
